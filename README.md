# ✅ Daily Habit Tracker

> **Build better habits. Stay consistent. Achieve your goals.**

A modern and feature-rich **Habit Tracking Web Application** built using **Python**, **Streamlit**, and **SQLite**. This application helps users create positive habits, monitor daily progress, maintain streaks, visualize performance through interactive analytics, and stay motivated with achievements and goals.
